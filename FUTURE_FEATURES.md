# Beast Tribe — Future Features (Post-Launch)

## Phase 2 Features

### 1. Calendar Sync
- Sync Beast Tribe events to device calendar (Apple Calendar, Google Calendar)
- When user joins an event, auto-add to their phone calendar with:
  - Event title, date, time
  - Location (if provided)
  - Reminder 1 hour before
- Use `expo-calendar` package
- Two-way sync: if user deletes from phone calendar, mark as "not going" in app
- Coach sessions auto-added to both coach and trainee calendars

### 2. Push Notifications
- New event in your area
- Someone beasted your post
- Pack chat messages
- Coach left you a note
- Event starting soon (30 min reminder)
- Streak about to break (evening reminder)
- Weekly Beast Score summary

### 3. Wearable Integration
- Apple Health / Google Fit sync for steps
- Heart rate during workouts
- Automatic workout detection
- Sleep tracking for recovery metrics

### 4. Advanced Coach Features
- Video workout demonstrations
- Live session streaming
- Automated progress reports (weekly PDF)
- Client compliance alerts
- Revenue tracking per coach

### 5. Social Features
- Direct messages between users
- @mentions in posts and comments
- Story-style content (24h expiry)
- Challenges between packs with leaderboard

### 6. Monetization
- Premium subscription tiers
- Coach marketplace (find and book coaches)
- In-app purchases (custom pack themes, badges)
- Event ticketing for paid workshops

### 7. AI Features
- AI workout suggestions based on history
- Smart nutrition recommendations
- Form correction via camera
- Predictive streak alerts
